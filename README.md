## Повежи руне
Игра меморије направљена као завршни пројекат (испитни задатак) курса Функционално програмирање на Универзитету Сингидунум

---

- Технологије коришћене на пројекту су HTML, CSS & JavaScript
- Пројекат је осмишљен тако да представи функционалну парадигму програмирања у реалном пројекту
- Ресурси пројекта су AI генерисани (слике самих руна)
